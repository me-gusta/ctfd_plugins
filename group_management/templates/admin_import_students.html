Import students from excel file

  1. Select rows from a table with four or three columns.
    (Last Name, First Name, Middle Name and student ID are required. Third column is optional)

  2. Copy selected rows to clipboard.

  3. Click on the input down below and paste.

  4. Press "Import" button.

Imported users will have their full name as a username and student ID as a password.
